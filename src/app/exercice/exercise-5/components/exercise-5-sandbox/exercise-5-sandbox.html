Composant legacy branche directement sur le service

Cette vue porte encore son etat local dans le composant et appelle directement `Exercise5ProductsService`. Le but est de migrer cette logique vers un NgRx Signal Store puis de rebrancher le composant.

Consigne

Point de depart : `src/app/exercice/exercise-5/components/exercise-5-sandbox/exercise-5-sandbox.ts`. Objectif : `src/app/exercice/exercise-5/components/exercise-5-sandbox/exercise-5-products.store.ts`.

  1. Reperez l etat, les derivees et les methodes dans le composant.
  2. Reperez aussi l appel direct a `Exercise5ProductsService`.
  3. Recreez cette logique dans le starter store avec `withEntities()`.
  4. Faites porter `loadProducts()` par le store plutot que par le composant.
  5. Branchez ensuite ce composant sur le store a la place du service.
  6. Comparez enfin votre resultat avec la correction finale.
