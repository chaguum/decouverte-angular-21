Service legacy a migrer vers un store

Cette vue est encore branchee sur un service local au composant. Le but est de migrer cette logique vers un NgRx Signal Store puis de rebrancher le composant.

Consigne

Point de depart : `src/app/exercice/exercise-5/components/exercise-5-sandbox/exercise-5-products.service.ts`. Objectif : `src/app/exercice/exercise-5/exercise-5-products.store.ts`.

  1. Reperez l etat, les derivees et les methodes dans le service.
  2. Recreez cette logique dans le starter store avec `withEntities()`.
  3. Ajoutez la garde anti rechargement dans `loadProducts()`.
  4. Branchez ensuite ce composant sur le store a la place du service.
  5. Comparez enfin votre resultat avec la correction finale.
