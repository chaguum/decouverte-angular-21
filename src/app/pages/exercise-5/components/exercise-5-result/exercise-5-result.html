Etat partage avec un NgRx Signal Store

Ici, le store est applicatif. Tant que l application SPA tourne, les produits deja charges et les favoris restent en memoire, meme si vous changez de page.

Ce que montre la correction

`withEntities()` porte la collection de produits, `withState` le reste de l etat brut, `withComputed` les derivees, `withMethods` les actions, `withHooks` le chargement initial et `withStatus()` le cycle `idle / pending / fulfilled / error`.
